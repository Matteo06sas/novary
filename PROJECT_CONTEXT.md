# NOVARY — PROJECT CONTEXT

## Mission

Novary è uno studio digitale premium specializzato nella realizzazione di siti web, e-commerce, applicazioni web e automazioni AI.

L'obiettivo non è creare semplici siti web, ma prodotti digitali moderni, veloci ed eleganti che aiutino aziende e professionisti a crescere online.

Ogni decisione progettuale deve essere orientata alla qualità, alla semplicità e alla scalabilità.

- riutilizzabile
- modulare
- semplice da mantenere

# Development Rules

NON ricreare il progetto.

NON modificare l'architettura se non richiesto.

NON sostituire componenti funzionanti.

Migliorare sempre il codice esistente.

Mantenere la struttura attuale.

Preferire modifiche incrementali.

---

# Quality Standard

Prima di ogni modifica chiediti:

"Questa modifica rende Novary più premium?"

Se la risposta è no,
non implementarla.

# Long-Term Vision

Novary non deve diventare soltanto un portfolio.

Diventerà una software house che sviluppa:

- siti web
- ecommerce
- SaaS
- automazioni AI
- dashboard
- gestionali
- applicazioni web
- prodotti digitali

Ogni progetto sviluppato dovrà poter essere riutilizzato come base per futuri clienti.

L'obiettivo finale è costruire una libreria proprietaria di componenti e template che riduca drasticamente i tempi di sviluppo mantenendo una qualità elevata.

Ogni decisione presa deve contribuire a questo obiettivo.
